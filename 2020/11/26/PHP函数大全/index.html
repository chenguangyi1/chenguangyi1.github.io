PHP函数大全

1.isset() 检查变量是否存在。

若变量不存在或为NULL，返回FALSE。

若变量不为NULL时，返回TRUE。

若变量为多个时，必须都不为NULL，才返回TRUE。

2.mysql_query(query,connection)执行一条MySQL查询。

query:(必需)规定要发送的SQL查询语句。注‘查询字符串不应以分号结束。

connection：(可选)SQL连接标识符。如果未规定，则使用上次的打开的连接。

如果之前没有打开连接，这个函数会尝试无参数调用mysql_conncet()来建立连接。

注：此函数仅对select,show,explain,describe句子返回资源标识符，如果查询失败则返回false。

对其他的SQL语句，执行成功则返回TRUE，失败则返回false。

3.mysql_fetch_array(result,resulttype)从结果集中取得一行作为关联数组，或数字数组，或二者兼有。

resulttype:(可选)规定要返回哪种类型的数组。

  • MYSQLI_ASSOC(关联数组)
  • MYSQLI_NUM(数字数组)
  • MYSQLI_BOTH(同时产生以上两者)–默认使用

4.mysql_query常常和mysql_fetch_array一起连用

例子：

$tempresult=mysql_query (“SQL语句”);
while ($row = mysql_fetch_array($tempresult))
{
print $row[0]; //打印第一列的记录
print $row[1]; //打印第二列的记录
}

数组变量 $row 被 mysql_fetch_array 函数建立并用查询的结果字段来填充，数组的每一个分量对应于查询结果的每一个字段 。

5.addslashes()和htmlspecialchars()的相同和区别

二者都用来都特殊符号进行转义，用于防止SQL语句的注入。

①addslashes()通常用于防止sql注入，它可对通过get，post和cookie传递过来的参数的单引号、双引号、\以及null前加“\”进行转义

如：如变量$str=$_POST[“str”];的值为：bb’ or 1=’1。通过addslashes()函数过滤后会变为：bb斜杆' or 1=斜杆'1。

②htmlspecialchars()也是对字符进行转义，与addslashes（）不同的是htmlspecialchars()是将特殊字符用引用实体替换。

③addslashes()在执行sql语句前对通过get、post和cookie传递来的参数中的单引号，双引号,\ 和null进行转义。但sql执行成功后，插入到数据库中的数据是不带有转义字符\的。这时如果插入到数据库中的是一些js脚本，当这些脚本被读取出来时还是会被执行。

这时我们可对读取出来的数据使用htmlspecialchars()进行过滤，避免执行被注入的脚本。

④在使用这两个函数之前需要先使用get_magic_quotes_gpc()验证下php.ini配置文件里的magic_quotes_gpc设置是否自动为GPC(get,post,cookie)传来的数据中的’”加上反斜线。如果没有确定直接使用该函数，可能造成双层转义。

6.get_magic_quotes_gpc()

用于检验php.ini配置文件中magic_quotes_gpc的设置是否开启自动为GPC(GET,POST,COOKIE)传来的数据进行转义。

7.ctype_digit()检测字符串中是否只含有数字。注：字符串里的数字，负数，小数都不行。

例子：

<?php
$strings = array(‘1820.20’, ‘10002’, ‘wsl!12’);
foreach ($strings as $testcase) {
if (ctype_digit($testcase)) {
echo “The string $testcase consists of all digits.\n”;
} else {
echo “The string $testcase does not consist of all digits.\n”;
}
}

例子：字符串里的数字才可以！！！

附：is_numeric() — 检测变量是否为数字或数字字符串(数字可以为负数，小数)

<?php

$numeric_string = ‘42’;
$integer = 42;

ctype_digit($numeric_string); // true
*ctype_digit($integer); // false (ASCII 42 is the * character)*

is_numeric($numeric_string); // true
is_numeric($integer); // true

8.mysql_real_escape_string() 转义 SQL 语句中使用的字符串中的特殊字符。

下列字符受影响：

  • \x00
  • \n
  • \r
  • \
  • ‘
  • “
  • \x1a

功能于addslashes一样，范围不一样！
